###ПРИМЕР###

**ВХОД:**  
Датасет в формате CSV с колонками:
['ФИО', 'email', 'телефон', 'сумма', 'адрес доставки', 'номер договора']


**ОЖИДАЕМАЯ ОЧИСТКА:**
- ФИО → `[NAME]`
- email → `[EMAIL]`
- телефон → `[PHONE]`
- сумма → оставить (если не конфиденциальна)
- адрес → `[ADDRESS]`
- номер договора → `[CONTRACT_ID]`

**ПРИМЕР PYTHON-КОДА:**
```python
import pandas as pd
import re

def redact_dataset(path_to_csv, output_path):
    df = pd.read_csv(path_to_csv)

    if 'ФИО' in df.columns:
        df['ФИО'] = '[NAME]'
    if 'email' in df.columns:
        df['email'] = df['email'].apply(lambda x: '[EMAIL]' if pd.notna(x) else x)
    if 'телефон' in df.columns:
        df['телефон'] = df['телефон'].apply(lambda x: '[PHONE]' if pd.notna(x) else x)
    if 'адрес доставки' in df.columns:
        df['адрес доставки'] = '[ADDRESS]'
    if 'номер договора' in df.columns:
        df['номер договора'] = '[CONTRACT_ID]'

    df.to_csv(output_path, index=False)
```
